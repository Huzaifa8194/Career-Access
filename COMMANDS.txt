CAREER ACCESS — Windows PowerShell commands
============================================

Prerequisites
-------------
- Node.js 20 LTS recommended, npm installed.
- (Optional for emulators & deploy) Firebase CLI:

   npm install -g firebase-tools


First-time setup
----------------
1) Install web app dependencies:

   npm install

2) (Optional) Install Cloud Functions dependencies:

   cd functions
   npm install
   cd ..


Environment variables (.env.local)
----------------------------------
Create a file named `.env.local` at the project root with:

   NEXT_PUBLIC_FIREBASE_API_KEY=AIzaSyDI5oB9Fdae7qBIRfuAKZwIZbQ3sUKg88U
   NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN=careeraccesshub-d3d27.firebaseapp.com
   NEXT_PUBLIC_FIREBASE_PROJECT_ID=careeraccesshub-d3d27
   NEXT_PUBLIC_FIREBASE_STORAGE_BUCKET=careeraccesshub-d3d27.firebasestorage.app
   NEXT_PUBLIC_FIREBASE_MESSAGING_SENDER_ID=969717593106
   NEXT_PUBLIC_FIREBASE_APP_ID=1:969717593106:web:f37927b0cfafbf22bc94cb

To also enable local emulators during dev, append:

   NEXT_PUBLIC_USE_FIREBASE_EMULATORS=true
   NEXT_PUBLIC_FIREBASE_EMULATOR_HOST=localhost


Run the dev server
------------------
3) Start the Next.js dev server (http://localhost:3000):

   npm run dev


Firebase login & project
------------------------
4) Log in to Firebase (one time):

   firebase login

5) Point the CLI to the project (already set in .firebaserc):

   firebase use careeraccesshub-d3d27


Deploy Firestore & Storage rules
--------------------------------
6) Deploy rules + indexes:

   firebase deploy --only firestore:rules
   firebase deploy --only firestore:indexes
   firebase deploy --only storage


Deploy Cloud Functions
----------------------
7) Build and deploy functions:

   cd functions
   npm run build
   cd ..
   firebase deploy --only functions


Run local emulators (recommended while developing)
--------------------------------------------------
8) Start Auth, Firestore, Storage, Functions emulators:

   firebase emulators:start

   Emulator UI:      http://localhost:4000
   Firestore port:   8080
   Auth port:        9099
   Storage port:     9199
   Functions port:   5001

   Keep `NEXT_PUBLIC_USE_FIREBASE_EMULATORS=true` in `.env.local` so the
   web app connects to the emulators while `npm run dev` is running.


Bootstrap an admin user
-----------------------
9) After you register an account at /portal/sign-up:

   a. Open the Firebase console -> Firestore -> users collection.
   b. Find your uid document and set   role: "admin"
      (Or use setUserRole callable function once another admin exists.)

Production build
----------------
10) Build and run production server:

    npm run build
    npm run start


Troubleshooting
---------------
- Firebase rules deploy fails with "Project not found":
     firebase use --add
  and choose careeraccesshub-d3d27.

- "Missing or insufficient permissions" at runtime:
  Confirm your `users/<uid>` document exists and that `role` is set to
  participant | advisor | admin.

- Functions build fails on deploy:
     cd functions
     npm install
     npm run build
     cd ..

- Want a fresh emulator DB each run:
     firebase emulators:start --import=./emulator-data --export-on-exit


Fix: "Firestore Database '(default)' not found"
-----------------------------------------------
If you see:
  Firestore (...) Database '(default)' not found

Do this in order:

1) Confirm you are on the correct Firebase project:

   firebase projects:list
   firebase use careeraccesshub-d3d27

2) Create Firestore database in Firebase Console (one-time, no CLI command):

   https://console.firebase.google.com/project/careeraccesshub-d3d27/firestore

   Click "Create database", choose:
   - Database ID: (default)
   - Mode: Production (or Test for quick local checks)
   - Region: pick the same region you plan to use for hosting/functions.

3) Restart your local app:

   npm run dev

4) (Optional) If you want to avoid production while testing, run emulators:

   In `.env.local` set:
      NEXT_PUBLIC_USE_FIREBASE_EMULATORS=true
      NEXT_PUBLIC_FIREBASE_EMULATOR_HOST=127.0.0.1

   Then run:
      firebase emulators:start
      npm run dev
