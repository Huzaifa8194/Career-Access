We are looking for a strong web designer/developer to build a new dedicated microsite and portal MVP for a serious adult education / workforce pathways initiative.

This is not a generic marketing website. It needs to look credible for:
- adult learners
- referral partners
- public-sector / grant-facing audiences
- future program operations

We already have an existing company website, but we do not want to modify it for this project. We want a new site or microsite with a clean public-facing experience plus a lightweight portal.

## Scope of Work

### Public-facing site
Build a clean, professional microsite with these pages:
- Home
- How It Works
- Services
- For Partners
- Get Started
- Contact

The site should communicate:
- adult pathways to college, training, and apprenticeship
- guided support from intake to enrollment
- services such as admissions support, FAFSA/financial aid guidance, short-term training pathways, apprenticeship advising, and enrollment follow-through
- partner/referral pathways for community organizations

### Portal MVP
Build a lightweight but believable portal with 3 user roles:
- Participant
- Advisor/Counselor
- Admin

#### Participant features
- registration/login
- intake / eligibility questionnaire
- dashboard with pathway, next steps, outstanding tasks, reminders
- checklist / task flow
- document upload
- appointment request
- messages / reminders

#### Advisor features
- participant pipeline dashboard
- participant list by stage
- participant profile with intake, notes, documents, and milestone tracking
- risk flags for stalled participants
- follow-up logging

#### Admin features
- reporting dashboard
- participant counts by stage
- pathway breakdown
- referral source breakdown
- advisor assignment
- basic content/task management

## Workflow logic
Participants should be routed into one of 4 tracks:
1. College + FAFSA
2. Short-term training
3. Apprenticeship / pre-apprenticeship
4. GED/HSE + transition planning

## Design direction
The site and portal must feel:
- clean
- serious
- trustworthy
- adult learner focused
- public-sector / workforce credible
- modern but not flashy
- mobile-friendly
- accessible

Please avoid:
- startup gimmicks
- cartoonish nonprofit design
- overdesigned animations
- generic corporate filler
- unnecessary complexity
